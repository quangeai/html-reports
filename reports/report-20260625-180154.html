二、 低偏差病例深度分析 低偏差病例通常指医疗费用与DIP支付标准较为接近，成本控制较好的病例。 ### 2.1 费用与成本分析 低偏差病例平均医疗总费用为 **1,783元**，平均DIP支付为 **740元**，平均成本为 **868元**，整体实现盈利（平均盈亏 **322元**）。这表明在费用较低、住院周期短（平均 **2.29天**）的病例中，科室能够有效控制成本，获得正向收益。 **费用结构饼图：** ```mermaid pie title 低偏差病例平均费用构成 "医技费 (检查/化验)" : 66.78 "药品费" : 10.16 "其他费用" : 22.14 "材料费" : 0.97 ``` *注：其他费用包含治疗、护理、床位、诊察等。* ### 2.2 病种分布 低偏差病例主要集中于轻症或常规诊疗病种： * **未特指的短暂性大脑缺血性发作 (G45.9)**：2例 * **椎基底动脉综合征 (G45.0)**：2例 * **脑梗死 (I63.900)**：1例 * **血容量减少性休克 (R57.1)**：1例 * **脑血管病:治疗性操作组 (S3:I60-I69-2)**：1例 ## 三、 亏损病例深度分析 亏损病例是本次分析的重点，其特点是医疗总费用高昂，但DIP支付率极低，导致严重亏损。 ### 3.1 亏损根源：支付率过低 亏损病例的平均支付率仅为 **22.18%**，远低于低偏差病例的 **57.95%**。这意味着医院花费了高昂的成本（平均 **7,548元**），但仅能通过DIP获得约 **1,732元** 的支付，成本覆盖严重不足。 **支付率与费用对比散点图：** ```mermaid xychart-beta title “亏损病例：支付率 vs 医疗总费用” x-axis “医疗总费用（元）” [9000, 9600, 17800, 18000, 18000, 23000] y-axis “支付率 (%)” [0.02, 2.46, 6.16, 25.14, 33.77, 34.09] bar [0.02, 2.46, 6.16, 25.14, 33.77, 34.09] ``` *图表显示，无论总费用高低，支付率普遍处于极低水平。* ### 3.2 费用结构异常：药占比畸高 亏损病例的平均药占比高达 **41.37%**，其中三例“未特指的脑梗死:血栓溶解药的注射或输注”病例的药占比更是超过 **50%**。这显著偏离了DIP病种分值的常规费用结构预期，可能是导致支付率被大幅调低（费用偏差过大）的关键原因。 **费用结构对比图：** ```mermaid pie title 亏损病例 vs 低偏差病例 费用结构对比 "亏损病例-药品费" : 41.37 "亏损病例-医技费" : 33.85 "亏损病例-其他" : 23.60 "低偏差病例-药品费" : 10.16 "低偏差病例-医技费" : 66.78 "低偏差病例-其他" : 22.09 ``` ### 3.3 主要亏损病种 亏损高度集中在少数重症病种，特别是涉及高值药品或复杂操作的病例： 1. **未特指的脑梗死:血栓溶解药的注射或输注 (I63.9+99.1000)**：3例，平均亏损 **-8,660万元**，平均药占比 **50.22%**。 2. **大脑动脉血栓形成引起的脑梗死 (I63.3)**：2例。 3. **脑干梗死 (I63.901)**：1例。 4. **大脑动脉未特指的闭塞或狭窄引起的脑梗死 (I63.5)**：1例。 ## 四、 核心发现与建议 ### 4.1 核心发现 1. **盈亏两极分化**：低偏差病例（轻症、短住院）盈利稳定；亏损病例（重症、长住院、高药耗）亏损严重，形成“以盈补亏”的局面。 2. **支付率是盈亏关键杠杆**：亏损病例的核心问题是实际费用结构（尤其是药占比）严重偏离DIP支付模型的预期，导致支付率被大幅降低。 3. **高值药品是主要风险点**：分析表明，**血栓溶解药**等高价药品的广泛使用，是导致药占比畸高、费用严重偏离、进而引发巨额亏损的最直接原因。 4. **病种结构影响显著**：亏损集中于“脑梗死”相关重症及操作组，这些病种本身资源消耗大，费用波动风险高。 ### 4.2 管理建议 1. **加强重症病种成本管控**： * 对“脑梗死”等亏损高发病种建立**临床路径和费用预警机制**。 * 重点监控**药占比**，特别是高值药品（如血栓溶解药）的合理使用，评估其临床必要性与经济性。 2. **优化费用结构**： * 分析DIP病种分值的费用构成标准，引导临床在诊疗过程中使实际费用结构（药、耗、技占比）向支付模型靠拢，避免因结构性偏差导致支付率打折。 3. **推进精细化管理**： * 将DIP分析数据反馈至临床科室和医疗组，树立成本意识。 * 区分“低偏差”和“亏损”病例进行差异化管理和考核。 4. **数据持续监测**： * 定期生成此类分析报告，动态跟踪各病种、各医师组的DIP执行情况。 * 对新出现的亏损病例进行个案复盘，及时识别并干预新的风险点。 ##
