系统登录功能测试用例集

用例ID 用例名称 详细描述 执行步骤 优先级 通过标准
TC-001 正常登录成功验证 输入正确的用户名和密码，验证系统登录流程正常 1. 打开系统登录页面
2. 在用户名输入框输入有效用户名（如testuser）
3. 在密码输入框输入匹配的正确密码（如test123）
4. 点击【登录】按钮		 高 成功跳转至系统首页，页面显示用户昵称等个人身份信息
TC-002 用户名空登录验证 用户名输入为空，密码输入正确，验证登录失败提示 1. 打开系统登录页面
2. 用户名输入框保持为空
3. 在密码输入框输入正确密码（如test123）
4. 点击【登录】按钮		 高 页面弹出“用户名不能为空”提示，停留在登录页面
TC-003 密码空登录验证 密码输入为空，用户名输入正确，验证登录失败提示 1. 打开系统登录页面
2. 在用户名输入框输入正确用户名（如testuser）
3. 密码输入框保持为空
4. 点击【登录】按钮		 高 页面弹出“密码不能为空”提示，停留在登录页面
TC-004 用户名正确密码错误登录验证 输入正确用户名和错误密码，验证登录失败提示 1. 打开系统登录页面
2. 在用户名输入框输入正确用户名（如testuser）
3. 在密码输入框输入错误密码（如wrong123）
4. 点击【登录】按钮		 高 页面弹出“密码错误，请重新输入”提示，停留在登录页面
TC-005 用户名不存在登录验证 输入不存在的用户名和任意密码，验证登录失败提示 1. 打开系统登录页面
2. 在用户名输入框输入不存在的用户名（如nonexistuser）
3. 在密码输入框输入任意密码（如123456）
4. 点击【登录】按钮		 高 页面弹出“用户名不存在，请注册”提示，停留在登录页面
TC-006 连续3次错误密码锁定账号验证 连续3次输入正确用户名和错误密码，验证账号锁定机制 1. 打开系统登录页面
2. 输入正确用户名（如testuser）和错误密码（如wrong1），点击【登录】按钮，确认提示“密码错误，请重新输入”
3. 重复步骤2两次，累计3次错误登录尝试
4. 第4次输入正确用户名和任意密码，点击【登录】按钮		 中 页面弹出“账号已锁定，请5分钟后重试”提示，账号无法登录
TC-007 账号锁定后5分钟解锁验证 账号锁定后等待5分钟，验证账号解锁后可正常登录 1. 按照TC-006步骤使账号锁定
2. 等待5分钟后，打开登录页面
3. 输入正确的用户名和密码，点击【登录】按钮		 中 成功跳转至系统首页，账号可正常使用
TC-008 记住密码功能验证 勾选记住密码后登录，验证下次登录自动填充账号密码 1. 打开系统登录页面
2. 输入正确的用户名和密码
3. 勾选【记住密码】复选框
4. 点击【登录】按钮，成功进入首页
5. 退出系统，重新打开登录页面		 中 登录页面自动填充之前输入的用户名和密码
TC-009 取消记住密码功能验证 取消记住密码后登录，验证下次登录不自动填充账号密码 1. 按照TC-008步骤使登录页面自动填充用户名和密码
2. 取消勾选【记住密码】复选框
3. 点击【登录】按钮，成功进入首页
4. 退出系统，重新打开登录页面		 低 登录页面未自动填充用户名和密码
TC-010 微信第三方登录验证 点击微信登录图标，验证第三方授权登录流程 1. 打开系统登录页面
2. 点击【微信登录】图标
3. 在跳转的微信授权页面，使用已绑定的微信账号完成授权		 高 授权成功后自动跳转至系统首页，页面显示用户个人信息
TC-011 QQ第三方登录验证 点击QQ登录图标，验证第三方授权登录流程 1. 打开系统登录页面
2. 点击【QQ登录】图标
3. 在跳转的QQ授权页面，使用已绑定的QQ账号完成授权		 高 授权成功后自动跳转至系统首页，页面显示用户个人信息
TC-012 第三方登录取消授权返回验证 第三方授权页面取消授权，验证返回登录页面 1. 打开系统登录页面
2. 点击【微信登录】或【QQ登录】图标
3. 在跳转的授权页面，点击【取消授权】按钮		 低 返回系统登录页面，未进入系统首页
