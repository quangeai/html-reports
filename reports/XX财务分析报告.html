五粮液股份公司下属公司2019年度财务分析报告

报告部门： 财务部

报告日期： 2020年1月

报告对象： 公司管理层、董事会审计委员会

报告范围： 合并报表范围内各子公司、分公司

编制依据： 2019年度合并财务报表及附注

一、公司基本情况

本报告分析主体为五粮液股份公司下属合并报表范围内的各经营单位。公司主营业务以白酒的生产、销售及相关配套业务为主。截至2019年12月31日，合并口径总资产为44.82亿元，较年初增长10.01%；所有者权益为16.97亿元，较年初增长9.16%；实现营业收入64.16亿元，同比增长11.11%；归属于母公司所有者的净利润为-468.45万元，同比由盈转亏。

从财务特征看，公司当前处于规模扩张与盈利承压并存的调整期。一方面，总资产与营业收入均实现增长，显示业务规模持续扩张；另一方面，净利润出现亏损，且未分配利润累计亏损额扩大至6.13亿元，已对所有者权益形成严重侵蚀。公司需高度关注高杠杆运营、盈利能力下滑、利息负担沉重等核心财务风险。

二、财务状况分析

（一）资产结构分析

1. 资产规模与增长

期末总资产4,481,606,462.75元，较年初4,073,804,089.35元增长10.01%，绝对额增加4.08亿元。资产增长主要驱动因素为：在建工程大幅增加3.13亿元（增长45.39%），以及应收账款增加9,198.33万元（增长23.73%）。

2. 资产结构及变动

期末流动资产占总资产比重为46.13%（年初46.65%），非流动资产占比为53.87%（年初53.35%），资产结构基本稳定，非流动资产占比略升。资产流动性整体一般，大量资金沉淀于长期资产。

3. 主要资产项目分析

  • 货币资金：期末余额5.24亿元，较年初增长34.31%，主要源于本年经营活动现金流改善及筹资活动现金净流入。但需关注其与短期借款的匹配性。
  • 应收账款：期末余额4.80亿元（不含应收票据），较年初增长23.73%，增幅高于营业收入增幅，表明销售回款压力增大，信用政策可能有所放松。账龄结构数据未提供，存在坏账风险。
  • 存货：期末余额7.27亿元，较年初增长4.33%，其中库存商品5.91亿元，占比较高。库龄结构数据未提供，需关注是否存在滞销或跌价风险。
  • 固定资产：净额108.20亿元，较年初减少5.75%。固定资产成新率（固定资产净值/固定资产原价）约为49.8%，资产老化程度较高，未来更新改造压力大。
  • 在建工程：期末余额10.03亿元，较年初激增45.39%，是资产增长的最主要来源。大规模在建工程转固后，将带来巨大的折旧压力，且项目收益存在不确定性。

（二）负债与权益结构分析

1. 资本结构

期末负债总额27.84亿元，所有者权益总额16.97亿元，产权比率（负债/权益）为1.64。公司财务杠杆较高，对债务融资依赖严重。

2. 负债结构

流动负债占比高达99.68%，负债期限结构高度短期化，面临较大的短期偿债压力。主要负债项目：短期借款8.88亿元（较年初下降22.83%），应付票据及应付账款10.01亿元（增长42.12%），其他流动负债4.84亿元（增长51.52%）。应付账款的大幅增加，占用了供应商资金，但也可能影响供应链关系。

3. 所有者权益

净资产增长主要源于实收资本增加3亿元（增资扩股）。未分配利润累计亏损高达6.13亿元，且本年度亏损进一步扩大，已严重侵蚀股东权益。资本公积略有减少，少数股东权益大幅减少，需关注少数股东权益变动原因。

（三）偿债能力分析

1. 短期偿债能力

流动比率 = 流动资产 / 流动负债 = 2,067,386,446.90 / 2,775,522,753.77 = 0.74
速动比率 = (流动资产 - 存货) / 流动负债 = (2,067,386,446.90 - 726,589,293.74) / 2,775,522,753.77 = 0.48

流动比率与速动比率均远低于通常认为的安全线（流动比率1.2，速动比率0.6），短期偿债能力严重不足，存在显著的流动性风险。

2. 长期偿债能力

资产负债率 = 负债总额 / 资产总额 = 2,784,452,753.77 / 4,481,606,462.75 = 62.13%
权益乘数 = 资产总额 / 所有者权益总额 = 4,481,606,462.75 / 1,697,153,708.98 = 2.64

资产负债率超过60%，处于较高水平。结合公司净利润亏损的现状，长期偿债能力堪忧，高杠杆运营模式风险极高。
