五粮液股份公司下属公司2019年度财务分析报告

一、公司基本情况

本报告分析主体为五粮液股份公司下属合并报表范围内的子公司及分公司（以下简称“公司”）。公司主营业务为白酒的生产与销售，属于食品饮料行业。截至2019年12月31日，公司总资产44.82亿元，较年初增长10.01%；所有者权益16.97亿元；全年实现营业收入64.16亿元，同比增长11.11%；净利润为-468.45万元，由盈转亏。公司当前处于业务扩张与结构调整并存期，资产规模持续增长，但在建工程投入巨大，同时面临盈利能力下滑、财务费用高企、短期偿债压力凸显等核心问题，需予以高度关注。

二、财务状况分析

（一）资产结构分析

1. 资产规模与增长

期末总资产为4,481,606,462.75元，较年初4,073,804,089.35元增长10.01%，增加4.08亿元。资产增长主要驱动因素为：在建工程增加3.13亿元（增长45.41%），以及应收账款增加0.92亿元（增长23.72%）。资产扩张速度较快，投资活动活跃。

2. 资产结构及变动

期末流动资产占比46.13%（2,067,386,446.90元），非流动资产占比53.87%（2,414,220,015.85元）。与年初相比，流动资产占比下降约1.5个百分点，非流动资产占比相应上升，资产结构呈现长期化、重型化趋势，资产流动性有所减弱。

3. 主要资产项目分析

  • 货币资金：期末余额5.24亿元，较年初增长34.32%，主要得益于经营活动现金流改善及筹资活动现金净流入。但需关注其是否受限。
  • 应收账款：期末余额4.80亿元（不含应收票据），较年初增长23.72%，增幅高于营业收入增幅（11.11%），表明销售回款速度可能放缓，信用政策有所放松或下游客户资金压力加大。结合附注，账龄结构数据未提供，坏账风险需评估。
  • 存货：期末余额7.27亿元，较年初增长4.33%。其中库存商品5.91亿元，占存货比重81.4%，金额较年初略有增加。需关注库存商品库龄及跌价准备计提是否充分。
  • 固定资产：账面净值108.20亿元，较年初下降5.77%。固定资产成新率（净值/原价）约为49.8%，设备老化程度需关注。
  • 在建工程：期末余额10.03亿元，较年初大幅增长45.41%，是资产增长的最主要来源。大规模的在建工程投入将带来未来巨大的折旧压力，且项目收益存在不确定性，需密切关注项目进度与资金保障。

（二）负债与权益结构分析

1. 资本结构

期末负债总额27.84亿元，所有者权益总额16.97亿元，产权比率（负债/权益）为1.64。公司财务杠杆较高，对债务融资依赖度大。

2. 负债结构

流动负债占比高达99.68%（27.76亿元），非流动负债仅占0.32%（893万元）。负债期限结构严重短期化，存在显著的短期偿债压力。主要负债项目：短期借款8.88亿元（较年初下降22.84%），应付票据及应付账款10.01亿元（大幅增长42.12%），其中应付票据激增101.23%。对上游供应商的资金占用增加，但同时也可能影响供应链关系。

3. 所有者权益

所有者权益增长主要来自实收资本增加3亿元（增资）以及资本公积的变动。未分配利润为-6.13亿元，亏损状态持续且本年有所扩大，已对所有者权益形成严重侵蚀。归属于母公司所有者权益中，未分配利润为-6.13亿元，表明公司历史累积亏损严重。

（三）偿债能力分析

1. 短期偿债能力

流动比率 = 流动资产 / 流动负债 = 2,067,386,446.90 / 2,775,522,753.77 ≈ 0.74。
速动比率 = (流动资产 - 存货) / 流动负债 = (2,067,386,446.90 - 726,589,293.74) / 2,775,522,753.77 ≈ 0.48。
两项指标均远低于通常认为的安全线（流动比率1.2，速动比率0.6），短期偿债能力严重不足，存在较大的流动性风险。

2. 长期偿债能力

资产负债率 = 负债总额 / 资产总额 = 2,784,452,753.77 / 4,481,606,462.75 ≈ 62.13%。
权益乘数 = 资产总额 / 所有者权益总额 = 4,481,606,462.75 / 1,697,153,708.98 ≈ 2.64。
资产负债率处于较高水平，结合公司净利润为负的情况，长期偿债能力亦显薄弱，利息保障倍数（数据未直接提供，但利息费用高企）低下。

三、经营成果分析

（一）盈利规模与增长

全年实现营业收入64.16亿元，同比增长11.11%，收入规模保持增长。但净利润为-468.45万元，上年同期为盈利183.12万元，由盈转亏，收入与利润增长严重背离。利润总额3013.05万元，同比下降0.25%，但所得税费用高达3481.50万元，导致税后亏损。
