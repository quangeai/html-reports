2. 数据清洗函数

2.1 remove_missing_values

该函数用于从DataFrame中删除包含缺失值的行或列，支持多种缺失值处理策略。 

def remove_missing_values(df: pd.DataFrame, axis: int = 0, threshold: float = 0.5) -> pd.DataFrame:
    """
    删除DataFrame中包含缺失值的行或列
    
    参数:
        df: pd.DataFrame
            输入的DataFrame
        axis: int, 可选
            处理轴，0表示行，1表示列，默认为0
        threshold: float, 可选
            非缺失值比例阈值，低于该阈值的行/列将被删除，默认为0.5
    
    返回:
        pd.DataFrame
            删除缺失值后的DataFrame
    
    示例:
        >>> df = pd.DataFrame({'A': [1, 2, None], 'B': [4, None, 6]})
        >>> remove_missing_values(df)
           A    B
        0  1.0  4.0
    """
    threshold_count = int(threshold * df.shape[1 - axis])
    return df.dropna(axis=axis, thresh=threshold_count)

注意：当axis=1时，threshold参数表示非缺失值的比例，而不是绝对数量。例如，threshold=0.75表示至少75%的非缺失值才保留该列。

2.2 fill_missing_values

该函数用于填充DataFrame中的缺失值，支持多种填充策略，如均值、中位数、众数和前向/后向填充。 

def fill_missing_values(df: pd.DataFrame, strategy: str = 'mean', 
                       fill_value: Any = None) -> pd.DataFrame:
    """
    填充DataFrame中的缺失值
    
    参数:
        df: pd.DataFrame
            输入的DataFrame
        strategy: str, 可选
            填充策略，可选值为'mean', 'median', 'mode', 'ffill', 'bfill', 'constant'，默认为'mean'
        fill_value: Any, 可选
            当strategy为'constant'时使用的填充值，默认为None
    
    返回:
        pd.DataFrame
            填充缺失值后的DataFrame
    
    示例:
        >>> df = pd.DataFrame({'A': [1, 2, None], 'B': [4, None, 6]})
        >>> fill_missing_values(df, strategy='median')
             A    B
        0  1.0  4.0
        1  2.0  5.0
        2  1.5  6.0
    """
    df_filled = df.copy()
    
    if strategy == 'mean':
        return df_filled.fillna(df_filled.mean(numeric_only=True))
    elif strategy == 'median':
        return df_filled.fillna(df_filled.median(numeric_only=True))
    elif strategy == 'mode':
        return df_filled.fillna(df_filled.mode().iloc[0])
    elif strategy in ['ffill', 'bfill']:
        return df_filled.fillna(method=strategy)
    elif strategy == 'constant':
        return df_filled.fillna(fill_value)
    else:
        raise ValueError("Invalid strategy. Choose from 'mean', 'median', 'mode', 'ffill', 'bfill', 'constant'")

3.1 normalize_data

    """
    对DataFrame中的数值列进行标准化或归一化
    
    参数:
        df: pd.DataFrame
            输入的DataFrame
        method: str, 可选

    返回:
        pd.DataFrame
            标准化后的DataFrame
    
    示例:
        >>> df = pd.DataFrame({'A': [1, 2, 3], 'B': [4, 5, 6]})
        >>> normalize_data(df, method='min-max')
             A    B
        0  0.0  0.0
        1  0.5  0.5
        2  1.0  1.0
    """
    df_normalized = df.copy()
    numeric_cols = df_normalized.select_dtypes(include=['int64', 'float64']).columns
