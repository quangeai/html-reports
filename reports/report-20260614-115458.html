# 知识卡片：西游记狮驼岭核心角色 - **分类**：文学经典/神话角色 - **定义**：《西游记》中盘踞狮驼岭、阻拦唐僧师徒取经的主要妖怪群体及其关联神佛。 - **核心要点**： 1. **青狮精**：文殊菩萨坐骑，曾大闹天宫 2. **白象精**：普贤菩萨坐骑，长鼻威力无穷 3. **金翅大鹏雕**：如来佛祖舅舅，神通广大 4. **小钻风**：巡山小妖，因巡山歌谣知名 - **背景与关联**： - 三妖均出身佛/仙体系（菩萨坐骑、佛亲），背景深厚。 - 占据狮驼国，是取经路上势力最大、最凶险的妖王组合之一。 - 最终均由各自主人（文殊、普贤、如来）收服，彰显“关系网”与因果。 - **来源**：用户提供信息整合，基于《西游记》原著第七十四至七十七回内容。
