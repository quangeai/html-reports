# 珠海市华普食品机械制造有限公司信用风险分析报告 ## 执行摘要 本报告旨在分析**珠海市华普食品机械制造有限公司**（以下简称“华普公司”）的信用状况，以评估其业务员提出的“每月月底结清上月货款，并授信额度20万元”的合作请求中存在的潜在风险。报告基于天眼查提供的截至2026年06月16日的企业信用报告（专业版）数据进行分析。 **核心结论：** 综合评估，该公司**整体信用状况尚可，但存在一定风险点**。建议采取**审慎合作**的态度，授信额度需严格控制，并加强过程监控。 ## 关键指标概览 - **企业状态**: 存续（成立日期：1999-03-22） - **注册资本/实缴资本**: 128万人民币 - **企业规模**: 小型 (S) - **人员规模**: 小于50人 (参保人数：24人，2025年报) - **税务评级**: A级 (2025年度) - **司法风险**: 存在3起历史司法案件，无当前失信、被执行、限高等严重风险。 - **经营风险**: 2023年曾因未按时年报被列入经营异常名录，已移出。 ## 详细数据分析 ### 1. 基础信息与经营稳定性 - **成立时间长**: 公司成立于1999年，经营历史超过27年，表明其具备一定的市场生存能力和行业经验。 - **股权结构集中**: 实际控制人**郑小敏**持股50%，股权相对集中，有利于公司决策，但也可能意味着公司治理对个人依赖较强。 - **近期股东变更频繁**: 报告显示在2026年1月和3月发生了两次投资人变更，有股东退出也有新股东进入。这可能需要关注公司内部股权结构是否稳定，以及变更背后的原因。 - **社保缴纳正常**: 2024-2025年参保人数稳定在23-24人，且五险齐全，显示公司用工相对规范。 ### 2. 司法与经营风险分析（风险识别重点） - **历史法律诉讼**: 共有3起司法案件，均为**买卖合同纠纷**。值得注意的是： - 其中一起（2018年）公司作为**原告**，向济源市飞龙食品有限公司追讨货款，案件金额高达**3,048,717元**，并已进入执行程序。这反映出公司曾面临大额应收账款回收困难的问题。 - 另一起（2021年）公司作为**被告**，涉及与供应商的买卖合同纠纷。 - **经营异常记录**: 2023年7月因“未按规定期限公示年度报告”被列入经营异常名录，虽已于当月补报后移出，但反映出公司内部管理（如法务或财务部门）可能存在疏忽。 - **无严重违法记录**: 目前无失信被执行人、限制消费令、行政处罚、严重违法等记录，这是积极的信号。 ### 3. 经营能力与资质分析 - **行政许可齐全**: 拥有9项行政许可，包括最新的经营范围变更备案，业务范围合法合规。 - **资质证书**: 持有有效的《排污许可证》（有效期至2030年），符合环保要求。 - **知识产权**: 拥有3项专利（均已终止），显示公司过去有一定技术研发活动，但近期无新的知识产权产出。 - **财务数据不透明**: 年报中资产、营收、利润等关键财务数据均选择“不公示”，无法直接评估其盈利能力和偿债能力。这增加了对其真实经营状况的判断难度。 ### 4. 法定代表人关联风险 - 法定代表人**邓锦文**同时在外投资并任职于其他两家机械公司（珠海佰利、珠海维特立）。这需要关注是否存在关联交易、资金占用或同业竞争的风险。 ## 风险评估与建议 ### 合作风险综合评价 | 风险维度 | 评估结果 | 说明 | | :--- | :--- | :--- | | **主体资格** | 低风险 | 公司合法存续，历史久。 | | **司法风险** | **中等风险** | 有买卖合同纠纷历史，尤其是作为原告的大额执行案件，提示其客户回款风险可能传导至上游供应商。 | | **经营风险** | 中低风险 | 曾有经营异常记录，但已纠正；无当前重大经营处罚。 | | **财务风险** | **较高风险** | 财务数据不公开，实缴资本仅128万，却请求20万信用额度（约占注册资本15.6%），需警惕其实际偿债能力。 | | **管理风险** | 中等风险 | 近期股东频繁变更，法定代表人有多家关联企业。 | ### 具体建议 1. **授信决策建议**: - **不建议直接批准20万元全额授信**。考虑到其注册资本规模、历史回款问题及财务不透明，初始合作应更为谨慎。 - **建议方案**: 可考虑设置一个**较低的初始信用额度（例如5-8万元）**，并约定“**月底结清上月货款**”的支付条款。合作3-6个月后，若付款记录良好，再根据实际交易规模和评估结果逐步上调额度。 2. **风险缓释措施**: - **强调支付条款**: 在合同中明确“每月月底结清上月全部货款”，并约定逾期支付的违约金。 - **要求提供担保**: 可尝试协商由法定代表人邓锦文或实际控制人郑小敏提供个人连带责任担保。 - **分期发货**: 对于大额订单，采取“货到付款”或“分批发货、分批付款”的方式，控制单次风险敞口。 3. **合作过程监控**: - **定期复查信用**: 每季度或每半年通过天眼查等平台复查该公司司法、经营风险变化。 - **关注付款习惯**: 严格监控其付款是否及时，一旦出现逾期，应立即收紧信用政策甚至停止发货。 ## 结论 珠海市华普食品机械制造有限公司是一家经营多年的小型制造企业，具备基本的经营资质和A级纳税信用，无当前重大违法违规记录。**然而，其历史买卖合同纠纷（尤其是大额应收款执行案件）、财务信息不透明、注册资本规模较小以及近期股权变动等因素，构成了合作的主要风险点。** **最终意见**：对于其提出的“月底结清+20万授信”请求，**存在合作风险，建议不予全额批准**。应采取“**小额试单、严格支付、逐步建立信任**”的策略，并落实相应的风险控制措施，以在开发客户的同时有效保障我方债权安全。 --- **珠海华普公司信用风险分析报告 © 2026**
