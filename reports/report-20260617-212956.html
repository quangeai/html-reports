用户登录功能测试用例集

用例ID 测试用例名称 详细描述 执行步骤 优先级 通过标准 关联需求
TC-001 正常登录成功 输入符合要求的正确用户名和密码，验证系统登录功能正常 1. 打开系统登录页面
2. 输入符合格式的正确用户名（如testuser123）
3. 输入符合格式的正确密码（如Test@123456）
4. 点击「登录」按钮		 高 成功跳转到系统首页，页面顶部显示当前登录用户的信息 REQ-001
TC-002 用户名错误登录失败 输入错误用户名和正确密码，验证登录失败提示 1. 打开系统登录页面
2. 输入错误用户名（如wronguser）
3. 输入正确密码
4. 点击「登录」按钮		 高 页面弹出提示「用户名或密码错误」，停留在登录页面 REQ-001
TC-003 密码错误登录失败 输入正确用户名和错误密码，验证登录失败提示 1. 打开系统登录页面
2. 输入正确用户名
3. 输入错误密码（如Wrong@123）
4. 点击「登录」按钮		 高 页面弹出提示「用户名或密码错误」，停留在登录页面 REQ-001
TC-004 用户名长度小于6位登录失败 输入长度为5位的用户名，验证格式校验 1. 打开系统登录页面
2. 输入5位用户名（如user1）
3. 输入正确密码
4. 点击「登录」按钮		 中 页面提示「用户名长度需为6-16位」，无法完成登录 REQ-002
TC-005 用户名长度大于16位登录失败 输入长度为17位的用户名，验证格式校验 1. 打开系统登录页面
2. 输入17位用户名（如testuser1234567890）
3. 输入正确密码
4. 点击「登录」按钮		 中 页面提示「用户名长度需为6-16位」，无法完成登录 REQ-002
TC-006 密码长度小于8位登录失败 输入长度为7位的密码，验证格式校验 1. 打开系统登录页面
2. 输入正确用户名
3. 输入7位密码（如Test123）
4. 点击「登录」按钮		 中 页面提示「密码长度需为8-20位，且包含大小写字母和数字」，无法完成登录 REQ-003
TC-007 密码无大写字母登录失败 输入无大写字母的密码，验证格式校验 1. 打开系统登录页面
2. 输入正确用户名
3. 输入无大写字母的密码（如test@123456）
4. 点击「登录」按钮		 中 页面提示「密码长度需为8-20位，且包含大小写字母和数字」，无法完成登录 REQ-003
TC-008 连续3次登录失败锁定账号 连续3次输入错误密码，验证账号锁定功能 1. 打开系统登录页面
2. 输入正确用户名+错误密码，点击登录，重复操作3次
3. 第3次点击登录后观察页面状态		 高 页面提示「账号已锁定，请10分钟后再尝试」，无法进行登录操作 REQ-004
TC-009 锁定状态下登录失败 账号锁定后输入正确用户名和密码，验证锁定限制 1. 账号处于锁定状态时，打开登录页面
2. 输入正确用户名和密码
3. 点击「登录」按钮		 中 页面提示「账号已锁定，请10分钟后再尝试」，停留在登录页面 REQ-004
TC-010 勾选记住密码后自动填充 登录时勾选记住密码，验证退出后再次登录的自动填充功能 1. 打开登录页面
2. 输入正确用户名和密码
3. 勾选「记住密码」复选框
4. 点击登录进入首页
5. 点击退出登录，返回登录页面		 中 登录页面自动填充之前的用户名和密码，「记住密码」复选框处于勾选状态 REQ-005
TC-011 用户名或密码为空登录失败 用户名或密码为空时，验证登录拦截 1. 打开登录页面
2. 用户名空+正确密码，点击登录
3. 正确用户名+密码空，点击登录
4. 用户名和密码都空，点击登录		 高 分别提示「用户名不能为空」或「密码不能为空」，无法完成登录 REQ-001
