# Markdown Formatting Guidelines Below are the standard rules for formatting content using Markdown: - For headings, use number signs (`#`), e.g. `# Heading Level 1`, `## Heading Level 2` - For list items, start with dashes (`-`), e.g. ```markdown - First list item - Second list item ``` - To emphasize text, wrap it with asterisks (`*`), e.g. *this text is italicized*, **this text is bold** - For code or commands, surround them with backticks (`` ` ``), e.g. run the `npm install` command - For quoted text, use greater than signs (`>`), e.g. > This is a blockquote, used for citing external content - For links, wrap the text in square brackets `[]`, followed by the URL in parentheses `()`, e.g. [Markdown Official Guide](https://www.markdownguide.org) - For images, use square brackets `[]` for the alt text, followed by the image URL in parentheses `()`, e.g. ![Sample landscape image](https://picsum.photos/800/400)
