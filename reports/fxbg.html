# Markdown Formatting Guide ## Headings Use number signs (`#`) to create headings. The number of `#` symbols indicates the heading level. Example: ```markdown # Heading 1 ## Heading 2 ### Heading 3 ``` ## Lists For list items, start with dashes (`-`). Example: ```markdown - First item - Second item - Third item ``` ## Text Emphasis To emphasize text, wrap it with asterisks (`*`). Example: ```markdown This is *emphasized text* within a sentence. ``` ## Code and Commands For code or commands, surround them with backticks (`` ` ``). Example: ```markdown Use the command `git status` to check your repository status. ``` ## Quoted Text For quoted text, use greater than signs (`>`). Example: ```markdown > This is a blockquote. > It can span multiple lines. ``` ## Links For links, wrap the text in square brackets `[]`, followed by the URL in parentheses `()`. Example: ```markdown Visit [OpenAI's website](https://openai.com) for more information. ``` ## Images For images, use square brackets `[]` for the alt text, followed by the image URL in parentheses `()`. Example: ```markdown ![Descriptive alt text](https://example.com/image.jpg) ``` ## Complete Example ```markdown # My Document Title ## Introduction This is a *sample document* demonstrating Markdown formatting. ### Key Features - Easy to read and write - Converts to HTML - Widely supported > "Markdown is intended to be as easy-to-read and easy-to-write as is feasible." - John Gruber For more details, visit the [Markdown Guide](https://www.markdownguide.org). ![Markdown Logo](https://markdown-here.com/img/icon256.png) Use `# Heading` for titles and `- item` for lists. ```
